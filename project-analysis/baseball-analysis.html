Baseball Analysis

Completed: March 2024 Tech Stack: Python, R

Project Overview

I wanted to do this project after a debate online about Freddie Freeman vs baseball players of the 90s and 2000s

The Challenge

Description of the problem you were trying to solve.

  • How does Freddie Freeman hits stack up vs other players?
  • How many Homeruns Freddie hit vs the other players
  • How many games did Freddie play on averagevs the other players

The Solution

I gathered data from Baseball Reference about the players that I choose to compare to Freddie Freeman.

Key Features

  • Average Hits compared to each other
  • Average Home Runs compared to each other
  • Average Games Played compared to each other

Lessons Learned

Freddie Freeman is a great player and is comparable to other players in terms of performance metrics.
